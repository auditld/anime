## Color Scheme (Dark Mode Default)

- **Background**: Dark blue-gray (#0b1726)
- **Cards**: Slightly lighter dark (#111827)
- **Primary/Accent**: Green (#16a34a)
- **Text**: Light gray/white (#f8fafc)
- **Borders**: Dark gray (#1e293b)
- **Muted**: Gray (#64748b)

## Responsive Behavior

### Mobile (320px - 640px)
- 2 columns grid for anime cards
- Stacked header with search below
- Touch-optimized buttons

### Tablet (641px - 1024px)
- 3-4 columns grid
- Side-by-side header elements
- Larger touch targets

### Desktop (1025px+)
- 5-6 columns grid
- Full horizontal header
- Hover effects enabled
- Wider video player

## To See the Actual UI

1. Set up the API server (following api.txt)
2. Configure `.env` with `NEXT_PUBLIC_API_URL`
3. Run `npm run dev`
4. Open http://localhost:3000

The UI will then display with real data from the API!
